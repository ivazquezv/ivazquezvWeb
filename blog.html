Encriptación de datos

Lanzamiento: 2024 · Producto

Encryptify

Presentamos Encryptify, una aplicación de escritorio desarrollada en Java Swing que permite cifrar y verificar integridad de ficheros multimedia y contenedores: audio, vídeo, imágenes, PDF e imágenes ISO. Encryptify combina algoritmos de cifrado clásicos y modernos y ofrece modos de uso gratuitos para estudiantes y licencias de pago para usos forenses y legales de hashing y encriptado.

  • Aplicación de escritorio construida en Java Swing, multiplataforma
  • Cifrado híbrido con librerías que soportan algoritmos clásicos y modernos
  • Soporte de formatos: audio, vídeo, imágenes, PDF e imágenes ISO
  • Hashes forenses y trazabilidad para uso legal y pericial
  • Modelo de licencias: gratuito para estudiantes; licencias de pago para uso forense/legal
  • Exportación de evidencias con metadatos y registros de proceso
Visitar Encryptify Descargar Licencias

Importante: Encryptify es una herramienta diseñada para labores educativas y profesionales. La versión gratuita cubre necesidades formativas para estudiantes; la licencia comercial/forense habilita funcionalidades avanzadas de hashing, exportación de evidencia y soporte para uso legal.
